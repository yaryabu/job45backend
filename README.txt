Маленький сервер на Go для демонстрации на мастерклассе в 45 школе

Для запуска нужно:
1. Установить Go: https://golang.org/dl/
2. Запустить сервер командой 'go run main.go'

Идеи для улучшения:
1. Добавить к человеку ещё одно поле "Фамилия"
2. Сделать ещё один handler, который будет удалять человека
3. Сделать ещё один handler, который будет возвращать всех людей из базы

* handler - это функция, которая обрабатывает информацию, когда пользователи заходят на какую-то URL (/createPerson, /findPerson, и т.д.)
